Transforms 是什么

torchvision.transforms 是 PyTorch 用于 图像预处理和数据增强的工具包。Transform 作为深度学习中处理图片的一个强大的工具包。

  • 目的：
    1. 数据增强（Data Augmentation）：增加数据多样性，防止过拟合
    2. 数据预处理（Preprocessing）：把图像转换为模型可接受的格式（Tensor、归一化等）
  • 常见用法场景：
    • 图像分类
    • 目标检测
    • 分割任务

数据增强（Data Augmentation）

旋转和翻转

色彩变换

  • 改变亮度、对比度、饱和度、色相
  • 增加模型对不同拍摄条件的鲁棒性

随机仿射（Affine）

  • 功能：
    • 随机旋转、平移、缩放、剪切
  • 用途：
    • 数据增强
    • 提高模型对仿射变换的鲁棒性

Compose（组合多个 transform）

  • Compose 是把多个变换组合起来顺序执行
  • 顺序很重要：
    • 数据增强（翻转、旋转、裁剪）通常在 ToTensor 前
    • Tensor 处理（归一化）必须在 ToTensor 后

Lambda（自定义变换）

  • 可以把自定义的图像处理函数加入 transform pipeline
  • 常用于特殊操作，如 gamma 校正、特殊归一化

示例：训练集和测试集 transform 区别

使用到 Dataset

  • transform 参数会在 getitem 时自动对每张图片进行处理
  • DataLoader 取出的就是已经处理好的 Tensor

小技巧与注意事项

  • 随机增强只用于训练集
  • ToTensor 必须在数据增强前最后执行
  • Normalize 均值和标准差要按通道顺序给出[R,G,B]
  • Compose 顺序不能乱，先几何增强，再 ToTensor，再归一化
  • 自定义 transform 可以用 Lambda 或继承 transforms
